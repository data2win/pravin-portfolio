05 — Research

NLP + Public Health Discourse

NGIA Final Project Paper · Co-authored with Shweata Hegde
Analyzing Vaccine Discourse in YouTube Comment Sections — A Study of COVID-19 Vaccination in India
Ashoka University Young India Fellowship · 2024–25 · GitHub Repository ↗
Analyzed 4,856 YouTube comments related to COVID-19 vaccination using the YouTube API and OpenAI's GPT-4 for aspect-based sentiment analysis and emotional tagging. Identified anger as the most dominant emotion; anti-vaccine narratives were strongly associated with fear and misinformation. Applied a mixed-method approach combining manual observation, API data extraction, and LLM processing to classify sentiments, emotions, and vaccination stances at scale.
